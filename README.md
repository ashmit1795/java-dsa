# Java DSA Learning Repository

This is a Java programming learning repository containing educational programs organized by lessons from the GeeksforGeeks (GFG) Java course. The repository focuses on fundamental programming concepts and data structures.

## Repository Structure

The repository is organized into lesson-based directories under `GFG Java/`:

- **L02 - Conditionals** (8 programs): Conditional logic, if-else statements, ternary operators
- **L03 - Loops** (20 programs): For loops, while loops, do-while loops, nested loops
- **L04 - Pattern Printing** (8 programs): Various pattern printing exercises using nested loops
- **L05 - Methods** (8 programs): Method creation, recursion, built-in Math methods
- **L06 - Arrays** (11 programs): Array operations, array manipulation, array algorithms

## Program Structure

Each program follows a consistent structure:
- Standalone Java class named `programN` where N is the program number
- Uses `java.util.Scanner` for user input
- Contains a `main` method as the entry point
- Includes comments describing the problem statement
- Closes the Scanner resource at the end

## Key Concepts Covered

### Conditionals
- Integer vs real number checking
- Conditional logic with multiple approaches
- Ternary operator usage

### Loops
- Series printing (arithmetic progressions)
- Loop control and iteration patterns
- Nested loop structures

### Pattern Printing
- Number patterns
- Star patterns
- Mixed patterns using nested loops

### Methods
- Static method creation
- Recursion basics
- Built-in Math library functions (sqrt, cbrt, pow, max, min, abs, round, ceil, floor)
- Factorial calculations
- Permutation and combination (nPr, nCr)

### Arrays
- Array input and output
- Array traversal
- Array manipulation (sum, product, negative elements)
- Array algorithms (rotation, reversal, merging, segregation)

## Development Notes

- Programs are compiled with `javac` and run with `java`
- Each program is self-contained and can be run independently
- Compiled `.class` files are present in the same directories as source files
- Programs use basic Java syntax suitable for beginners

## File Naming Convention

- Source files: `programN.java` (where N is the program number)
- Compiled files: `programN.class` (generated automatically)
- Programs are numbered sequentially within each lesson directory

## Learning Path

The repository follows a progressive learning path:
1. **Conditionals**: Basic decision-making
2. **Loops**: Iteration and repetition
3. **Pattern Printing**: Applying loops for visual patterns
4. **Methods**: Code organization and reusability
5. **Arrays**: Data structure fundamentals

## Usage

To run any program:
```bash
javac "GFG Java/LXX - Topic/programN.java"
java "GFG Java/LXX - Topic/programN"
```

Or directly:
```bash
java "GFG Java/LXX - Topic/programN"
```
